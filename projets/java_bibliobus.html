Client lourd bibliobus

Dans le cadre de l'épreuve E6 de ma formation en BTS SIO option SLAM, il nous à été demander réaliser un client lourd en java visant à gérer une base de donnée. Via cette application, il est possible d'ajouter, de modifier ou de supprimer des livres d'une base de données

Page Github

La fenêtre ci-dessous permet d'ajouter des livres à la base de données, rafraîchir l'affichage et d'accéder au fenêtre de gestion des auteurs et des collections

En cliquant sur le bouton gérer les auteurs, nous somme rediriger sur une fenètre permettant de supprimer,modifier ou d'ajouter un auteur

En cliquant sur le bouton gérer les collections, nous somme rediriger sur une fenètre permettant de supprimer ou d'ajouter une collections

En cliquant sur le bouton gérer les éditeurs, nous somme rediriger sur une fenètre permettant de supprimer ou d'ajouter une collections
